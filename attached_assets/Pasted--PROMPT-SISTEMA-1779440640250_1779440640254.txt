## STACK OBRIGATÓRIA

Frontend   : Angular (standalone components) + Angular Material ou Bootstrap
Backend    : PHP puro — sem qualquer framework (sem Laravel, Slim, etc.)
Base Dados : MySQL
Comunicação: HTTP REST com JSON
Auth       : JWT (JSON Web Tokens)


## REQUISITOS DE AUTENTICAÇÃO

- Registo de utilizador com validação completa (nome, email, senha, confirmar senha)
- Login com geração de JWT
- Logout com invalidação de token no cliente
- Recuperação de senha por email (link com token temporário)
- Dois roles distintos: admin e cliente
- Guards Angular para proteger rotas privadas
- Interceptors para anexar JWT em todas as requests HTTP


## CRUD COMPLETO (backend + frontend)

VIAGENS:
  - Criar, listar, editar, eliminar viagens
  - Upload de imagem de capa
  - Campos: destino, data início, data fim, descrição, orçamento, estado, imagem

RESERVAS:
  - Criar, listar, editar, cancelar reservas
  - Estados: pendente / confirmado / cancelado
  - Relação com viagem e utilizador

UTILIZADORES:
  - Admin gere todos os utilizadores (listar, editar role, desactivar, eliminar)
  - Cliente gere apenas o seu próprio perfil (foto, dados, senha)


## PERMISSÕES

ADMIN:
  - Acesso total a utilizadores, viagens, reservas, relatórios e logs
  - Pode alterar o role de qualquer utilizador
  - Vê estatísticas globais do sistema

CLIENTE:
  - Vê e gere apenas as suas próprias viagens e reservas
  - Pode editar o seu perfil
  - Não tem acesso ao painel administrativo


## BASE DE DADOS

- Mínimo 3 tabelas com relações entre si
- Foreign keys e índices correctamente definidos
- Scripts SQL completos com comentários explicativos
- Seeds de dados de teste realistas (utilizadores, viagens, reservas)
- Sugestão de tabelas: users, trips, bookings (+ tabelas auxiliares se necessário)


## INTEGRAÇÕES COM APIs EXTERNAS

1. OpenWeather API
   - Consultar clima actual do destino de cada viagem
   - Mostrar temperatura, condição e ícone no detalhe da viagem e no dashboard

2. REST Countries API (restcountries.com)
   - Pesquisa de países e cidades ao criar/editar viagem
   - Mostrar bandeira, capital, fuso horário e moeda local

## QUALIDADE DO CÓDIGO
